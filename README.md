# 🌐 All About Chris

All About Chris is my personal portfolio website — a modern, responsive, and interactive showcase of my work, skills, and projects as a software developer.

Built with React, Vite, and powered by Bun, the project focuses on speed, simplicity, and clean UI powered by Tailwind CSS and React Bits.

⸻

## 🚀 Tech Stack
	•	⚛️ React – Frontend library for building UI
	•	⚡ Vite – Lightning-fast build tool
	•	🍞 Bun – Package manager and runtime used instead of npm
	•	🎨 Tailwind CSS – Utility-first CSS framework for styling
	•	🧩 React Bits – Pre-built, reusable UI components
	•	📩 Web3Forms – For handling form submissions securely
	•	💫 Tailwind Animations – Smooth, modern animations throughout the site
	•	☁️ Vercel – Hosting and deployment
	•	🐙 GitHub – Version control and collaboration

⸻

## 🧠 Features
	•	Fully responsive design for all devices
	•	Animated UI with Tailwind transitions
	•	Contact form integrated with Web3Forms
	•	Fast build and load time with Bun + Vite
	•	Clean and reusable component structure

  📦 Deployment

This project is deployed on Vercel.
Every push to the main branch triggers an automatic deployment from GitHub.
